# Vedio Editor Portfolio

A polished Next.js portfolio website that blends cinematic editor energy with smooth UX, motion, and modern frontend design.

## ✨ What this project showcases

- Creative video-editor inspired portfolio layout
- Animated hero sections, scroll reveals, and motion accents
- Reusable card components and section-driven design
- Responsive styling for desktop and mobile
- Built with the latest Next.js App Router

## 🚀 Why it’s different

This portfolio is designed to feel like a visual walkthrough, not just a static page. Every section is crafted to highlight skills, services, and projects with strong emphasis on storytelling and polished UI.

## 🧩 Core features

- Next.js 16 app router structure
- TypeScript-based components
- Smooth animations and section transitions
- Dedicated data files for projects, services, and testimonials
- Custom hooks for active section tracking and smooth scroll

## 🔧 Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser to view the portfolio.

## 📁 Project structure

- `src/app/` — layout and page entry
- `src/components/` — reusable UI building blocks
- `src/data/` — content-driven project and profile data
- `src/hooks/` — custom React hooks for interactivity
- `public/` — static image and video assets

## 💡 Customization

Edit the data files under `src/data/` and replace `public/` media to personalize the portfolio. The layout is designed to adapt to updated brand content quickly.

## 🌐 Deployment

Deploy to Vercel, Netlify, or any Next.js-compatible hosting provider.
